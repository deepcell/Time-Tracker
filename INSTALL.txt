Create a MySQL username and password.

Create a MySQL database. e.g. time

mysql> create database time;
mysql> quit;

Import the schema.

$ mysql -u username -p time < time.mysql

Edit config.class.php. Update the MySQL information.

        // MySQL
        $this->mysqlServer   = "localhost";
        $this->mysqlPort     = 3306;
        $this->mysqlUsername = "username";
        $this->mysqlPassword = "password";
        $this->mysqlDatabase = "time";

It is all done!
